# AI ReelStroller

AI ReelStroller is a **Next.js-based web application** that allows users to browse short video reels in a smooth, scroll-based interface.  
The project focuses on modern UI, performance, and basic AI-driven logic for reel ordering and interaction.

---

## Features
- Smooth reel scrolling experience
- Clean and responsive UI
- Component-based architecture using Next.js App Router
- Basic AI / logic-based reel filtering or ordering
- Optimized performance with modern Next.js features

---

## Tech Stack
- Next.js
- React
- JavaScript
- HTML
- CSS
